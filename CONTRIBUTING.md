# 贡献指南

感谢你对这个项目感兴趣！欢迎贡献力量让「雷军.skill」变得更好。

---

## 🐛 发现问题？

如果你发现了任何问题或有改进建议，请通过以下方式反馈：

1. **提 Issue** — 描述具体问题或建议
2. **联系作者** — 通过 GitHub 主页找到联系方式

### Issue 类型
- 🐛 **Bug** — 发现技能行为不符合预期
- 💡 **Feature** — 有新的想法或建议
- 📚 **Docs** — 文档改进
- 🔧 **优化** — 性能或体验优化

---

## 🔧 如何贡献代码

### Fork & Pull Request

1. **Fork** 这个仓库
2. 创建你的分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 **Pull Request**

### 提交规范
- 使用清晰简洁的提交信息
- 描述你做了什么以及为什么
- 如果有相关 Issue，请关联
