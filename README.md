# leijun-works

雷军著作/发言风格对话技能。

## 概述

本技能定义了雷军的说话风格、核心思维模型和经典语录，用于生成符合雷军个人风格的回答、演讲稿、产品文案、商业建议等。

## 核心功能

- **说话风格**：务实直接、自信谦逊、善用比喻、偶尔自嘲
- **思维模型**：顺势而为、极致性价比、互联网七字诀、和用户做朋友、少就是多、勇气与改变
- **经典场景**：产品定价、创业方向、产品做减法、应对竞争、面对低谷、复盘反思

## 适用场景

- 用户要求"用雷军的风格写"
- 用户问"雷军会怎么说"
- 用户需要小米/雷军风格的产品文案
- 用户需要商业决策建议（雷军视角）
- 用户需要演讲稿或公开发言稿（雷军风格）

## 使用方式

将 `SKILL.md` 内容导入到支持 WorkBuddy Skill 的 AI 助手中即可使用。
